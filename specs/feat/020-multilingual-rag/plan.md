# Implementation Plan: Multilingual RAG Q&A

**Branch**: `feat/020-multilingual-rag` | **Date**: 2026-04-01 | **Spec**: [spec.md](../../feat-020-multilingual-rag/spec.md)  
**Input**: Feature specification from `/specs/feat-020-multilingual-rag/spec.md`

## Summary

Add multilingual support to the existing English RAG Q&A pipeline so users can ask legal questions in isiZulu, Sesotho, or Afrikaans and receive answers in their language. A new `LanguageAppService` detects the input language via GPT-4o and translates non-English questions to English for knowledge-base search. The `RagPromptBuilder` gains a language directive that instructs the LLM to respond in the detected language while keeping Act names and section numbers in English. No new database migration is required — the `Questions` table already has `OriginalText`, `TranslatedText`, and `Language` columns.
